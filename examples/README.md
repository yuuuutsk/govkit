# govkit サンプルデータ

このディレクトリには、govkit の各ツールのサンプルデータと出力例が格納されています。

## egov-viewer サンプル

### sample-data/

e-Gov 通知書ビューワーのサンプル入力データです。個人情報はすべて架空のものです。

**ファイル構成：**

- `通知書.xml` - e-Gov 通知書本体のサンプル
- `保険料増減内訳書.xml` - 保険料増減内訳書のサンプル
- `納付額詳細.csv` - CSV データのサンプル（Shift-JIS エンコーディング）

### sample-output.html

上記サンプルデータから生成された HTML 出力例です。ブラウザで開いて実際の表示を確認できます。

**表示内容：**

- 通知書の基本情報（文書番号、日付、発信者など）
- 本文とお知らせ
- 保険料増減内訳書（個人別明細と集計）
- CSV データの表形式表示（行列反転機能付き）

### サンプルデータの利用方法

```bash
# サンプルデータから HTML を生成
govkit egov-viewer examples/sample-data

# 生成された output.html を開く
open output.html
```

## 注意事項

- すべてのサンプルデータは架空のものです
- 実際の e-Gov データとは形式が異なる場合があります
- 個人情報や機密情報は含まれていません
